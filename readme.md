# Como rodar
Para executar os comandos é preciso ter o python e os arquivos do requirement.txt instalados, assim como um npm.

- Execute o html,css e js dentro de um servidor

```bash
npx http-server
```

- Execute o servidor Python
```bash
python3 imagenet.py
```
